Installing Eclipse

  1. Get Eclipse from our Website or from the Eclipse Website. When using the Eclipse homepage you have to choose Eclipse MDT (Helios SR1 release or higher). If you want to create your models with Papyrus, Windows users need to download the 32-bit version. The 64-bit version still has problems modeling with Papyrus.
  2. Unpack Eclipse.


Installing CARiSMA

  1. Start Eclipse.
  2. Select the menu item Help → Install new Software...
  3. Use the CARiSMA Update Site.




  4. Select the desired Features.
    Note: A detailed description about the Features and the included Checks can be found on the CARiSMA website.
  5. Click Next twice.
  6. Accept the Licenses and click Finish.
  7. Restart Eclipse.
